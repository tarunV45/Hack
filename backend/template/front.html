Plant Disease Prediction

Input file path of your image file of your crop and retrieve the disease description and the supplements needed for the disease as a response in json format.

Request:

http://127.0.0.1:5000/plantdisease?filepath={filePath}

Response:

{
   "classname":{classname}
   "description":{description}
   "supplements":{supplements}
}
